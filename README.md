# TodoApp

Basit bir yapılacaklar listesi (To-Do List) uygulaması. Kullanıcılar görev ekleyebilir, güncelleyebilir ve silebilir.

## 🚀 Özellikler

- Görev ekleme
- Görev silme
- Görev tamamlama
- MySQL veritabanı entegrasyonu
- RESTful API

## 📦 Kullanılan Teknolojiler

- Java 17+
- Spring Boot
- Spring Data JPA
- MySQL
- Maven
- Lombok
- Swagger
